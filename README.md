# 3D 虚拟试衣数字人

## 项目概述

构建一个基于多 Agent 协作的 3D 虚拟试衣数字人系统。用户上传衣服照片后，AI 自动识别并提取服装特征，将其映射到 3D 数字人模型上，用户可在浏览器中自由旋转、缩放数字人，查看服装上身效果。

## 核心痛点

传统电商只能展示平面模特图，用户无法感知衣服穿在自己身上的真实效果，导致在线购衣退货率居高不下。本项目通过 AI + 3D 渲染技术，让用户在浏览器中即可完成虚拟试穿，降低购买决策风险和退货成本。

## 核心逻辑流

系统采用长链推理与多 Agent 协作架构，分为以下模块：

- **视觉理解 Agent** — 接收用户上传的衣服照片，完成服装分割、属性识别（款式、颜色、材质、领型等），将 2D 服装图像提取为结构化数据。
- **3D 重建 Agent** — 将提取的服装纹理映射到 3D 数字人模型上，通过 UV 展开和纹理投影处理正反面贴图，利用生成式模型补全遮挡区域和背面纹理。
- **渲染交互 Agent** — 基于 Three.js 构建 Web 端 3D 展示层，实现数字人的实时旋转、缩放，支持多角度查看服装上身效果。
- **协调 Agent** — 负责上述 Agent 之间的任务调度、结果校验与异常回退，确保端到端推理流程稳定高效。

## 技术栈

- **前端渲染**：React + Three.js（glTF/GLB 3D 模型加载与交互）
- **AI 视觉**：多模态大模型（服装分割与属性识别）
- **纹理映射**：UV 展开 + 生成式纹理补全
- **Agent 编排**：长链推理 + 多 Agent 协作调度
