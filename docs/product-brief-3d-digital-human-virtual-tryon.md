# Product Brief: 3D 数字人虚拟试穿系统

## Executive Summary

为电商平台构建可交付的 3D 数字人虚拟试穿系统。用户上传一张照片即可生成自己的 3D 数字分身，在线试穿服装和鞋子，360° 查看上身效果。系统作为一个完整项目交付给电商平台客户，支持私有化部署，覆盖国内和国际市场，男女品类皆适配。

核心策略是利用和改造开源 AI 模型（LHM、FASHN VTON），以最低的基础设施成本实现可商用的试穿效果，用工程整合能力而非自研算法形成壁垒。

## The Problem

线上服装零售面临结构性痛点：
- **退货率高企**：线上退货率达 19.3%（NRF 2025），其中"不合身/效果不符"是首要原因，美国年退货总额达 $850B
- **用户决策盲区**：传统电商只有平面模特图，买家无法感知衣服穿在"自己身上"的效果，"bracketing"（买多件退多件）成为常态
- **平台竞争升级**：Google 已将 AI 试穿集成到搜索结果（2026年4月），Shopify 集成了 Genlook，Zara 2025年底上线试穿——虚拟试穿正从实验变为竞争必需品

中国电商和跨境电商平台对虚拟试穿有强烈需求，但市面方案要么是 SaaS 按次收费（长期成本不可控），要么是闭源黑盒（无法定制），存在"可私有化部署 + 低成本 + 可定制"的市场空白。

## The Solution

一套完整的 3D 虚拟试穿系统，作为项目交付给客户平台：

1. **3D 数字人生成**：用户上传单张照片 → LHM 模型秒级生成全身 3D 数字人（SMPL-X 骨骼），也支持手动调参微调
2. **AI 虚拟试穿**：用户选择服装商品 → FASHN VTON 将服装迁移到数字人身上 → 360° 旋转查看各角度效果
3. **动画与场景**：Mixamo 预设骨骼动画（走路/转圈/挥手）+ 多场景切换（商场/沙滩/办公室），增强沉浸感
4. **Web 端交付**：基于 Three.js + LAM_WebRender 高斯泼溅渲染，浏览器即开即用，手机端可达 120FPS

一次交付，客户私有化部署，无持续 API 调用费用，且源码可定制。

## What Makes This Different

| 维度 | 市面方案 | 我们 |
|------|---------|------|
| 交付方式 | SaaS 按调用收费 | **一次性项目交付 + 私有化部署** |
| 模型 | 闭源自研/商业 API | **开源改造 + 工程优化**，不绑特定厂商 |
| 成本结构 | 持续 API 费用，量越大越贵 | GPU 自有，边际成本仅电费和硬件折旧 |
| 可定制性 | 黑盒，难以修改 | 源码交付，客户可二次开发 |
| 品类 | 多聚焦女装上衣 | 男女皆覆盖，上下装+鞋子 |
| 渲染 | 多数 2D 效果图 | **Web 3D 高斯泼溅**，手机 120FPS |

核心壁垒不是算法，而是**高质量开源模型的工程化整合能力**——把 LHM + FASHN VTON + LAM_WebRender + SMPL-X 这一整套管线调通、优化、产品化，降低客户的 AI 基础设施门槛。

## Who This Serves

**直接客户**：国内电商平台（淘宝/京东体系外的垂直电商、品牌电商）和跨境电商平台（SHEIN/Temu/独立站生态），希望拥有虚拟试穿能力但不愿绑定 SaaS 订阅。

**终端用户**：平台上的消费者，20-40 岁男女，网购服装频次高，因退货经历有决策焦虑。

**关键使用场景**：
- 买家选衣服 → 点"试穿" → 5 秒看效果 → 决策买不买
- 卖家上新品 → 用统一数字人展示多款服装 → 降低模特拍摄成本

## Success Criteria

| 指标 | 目标 |
|------|------|
| 项目交付客户数（首年） | 3-5 个 |
| 客户平台退货率降低 | ≥ 3 个百分点 |
| 试穿到加购转化率 | ≥ 15% |
| 数字人生成耗时 | < 5 秒 |
| 试穿效果生成耗时 | < 8 秒 |
| 单次试穿 GPU 成本 | < ¥0.05 |
| 手机端渲染帧率 | ≥ 25 FPS |

## Scope

### In Scope (v1)
- 照片数字人生成（LHM）
- 手动参数微调（身高/体重/肩宽/胸围/腰围/臀围/腿长）
- 上装 + 下装 + 鞋子试穿
- 360° 旋转查看 + 缩放
- 预设骨骼动画（4 个）+ 场景（5 个）
- 私有化部署方案 + 运维文档

### Out of Scope (v1)
- AR 实时摄像头试穿
- 多人同时试穿/社交分享
- 短视频生成
- AI 尺码推荐
- 移动端原生 App（浏览器即可）

## Competitive Landscape

| 类型 | 代表 | 弱点 |
|------|------|------|
| 大厂方案 | Google AI Try-On、Amazon | 不卖项目，绑定自身生态 |
| SaaS 方案 | DRESSX、Vue.ai、Zyler | 按次收费，长期成本高 |
| AR 方案 | Snap AR、ZOZO | 侧重虚拟量体，非真实试穿 |
| 开源项目 | OOTDiffusion、IDM-VTON | 单点方案，不成体系，缺交付能力 |

## Vision

如果 v1 跑通 3-5 个客户，v2 扩展方向：
- 增加配饰（眼镜/帽子/包包）试穿
- AI 尺码推荐（根据数字人体型 + 服装尺码表）
- 多语言国际化交付标准
- 商品视频生成（数字人穿衣服走秀）

长期目标：成为电商虚拟试穿领域的"WordPress"——开源基础版免费提供，商业交付版 + 定制服务收费。
